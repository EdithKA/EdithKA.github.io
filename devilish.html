Cutout Village

Interaction systems for Devilish Games.

Unity · C# Interaction Systems UX Programming Internship
Context

From brief to production

Devilish Games
Spanish indie studio · June 2 – July 31, 2025

Cutout Village is a simulation game about assembling paper model houses to revitalize a rural village — crafts meets management. During my degree internship I was assigned to build the entire assembly interaction system from scratch in Unity.

My contributions covered the full programming pipeline: tool mechanics (scissors, hand, glue), path-based movement using sphere markers, visual feedback states, step sequencing with a custom Step / StepGroup class architecture, camera controller, and cursor integration. I also cleaned and documented all code using Doxygen.
